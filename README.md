# Git Экспресс — Мобильное приложение для изучения Git

> **Версия:** 1.0
> **Платформа:** Android (min SDK 24)  
> **Тип сборки:** Release APK

Приложение разработано в рамках дипломного проекта для обучения сотрудников основам системы контроля версий Git. Оно сочетает в себе теоретические материалы, интерактивное тестирование и геймифицированные мини-игры для закрепления навыков.

---

## 🚀 Установка и запуск

Поскольку приложение распространяется в виде APK-файла, установка производится вручную:

1. **Скачайте файл** `git-express-app.apk` из этого репозитория.
2. **Разрешите установку** из неизвестных источников:
   - *Настройки → Безопасность → Неизвестные источники* (или следуйте подсказкам системы при открытии файла).
3. **Запустите установщик** и дождитесь завершения процесса.
4. Откройте приложение **Git Экспресс** на рабочем столе.

> ⚠️ **Важно:** Приложение не требует подключения к интернету и работает полностью в офлайн-режиме. Все данные сохраняются локально на устройстве.

---

## ✨ Основные возможности

### 🎮 Геймификация
- **Мини-игры:** 
  - расставь команды в правильном порядке.
  - сопоставь команду с её описанием.
  - классическая игра на память с терминами.
- **Система достижений:** Разблокируйте награды за идеальное прохождение тестов и секретные действия в интерфейсе.

### 🎨 Персонализация
- **Магазин тем:** Покупайте уникальные цветовые палитры за очки, заработанные в обучении.
- **Тёмный режим:** Полная поддержка тёмной темы для комфортного чтения в любое время суток.
- **Режим для слабовидящих:** Функция увеличения масштаба текста.
